## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

Ensure that you have mongo running (either locally or through docker) with a replica set enabled.

Ensure that there is an environment variable named `DATABASE_URL` which points to your local mongo database.

For eg., `DATABASE_URL="mongodb://localhost:27017/user-mgmt"`

Refer [this](https://gist.github.com/davisford/bb37079900888c44d2bbcb2c52a5d6e8#file-setup-mongodb-on-localhost-as-replica-set) to set up a replica set if you are running Mongo through `brew`

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
